# Repository Layout

- `.claude/` contains Claude tooling configuration and skills.
- `CLAUDE.md` contains workspace-specific agent instructions.
- `PRDs/` contains project requirements and planning documents.
- `Oleh-Hladyshev-Profile/` contains Oleh Hladyshev's React/Vite profile app.
- `Viktoriia-Lysenko-Profile/` contains Viktoriia Lysenko's React/Vite profile app.

## Apps

Run frontend commands from the repository root using a relative path.

Start Oleh's app with:

```bash
cd ./Oleh-Hladyshev-Profile
npm run dev
```

Start Viktoriia's app with:

```bash
cd ./Viktoriia-Lysenko-Profile
npm run dev
```

Other useful commands for either app:

```bash
cd ./<app-directory>
npm run build
npm run preview
npm run lint
```
